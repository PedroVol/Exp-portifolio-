Aulas de Experiência Criativa

A base de todo o portifólio é a diciplina de experiência criativa, muito por conta da utilização de html e a criação e desenvolvimento de web sites.

*No semestre tivemos 3 principais projetos que abordaram 3 linguagens diferentes:

Projeto 1: jogo

Um projeto feito em grupo usado Python e o aplicativo Construtc 3, nesse trabalho tivemos liberdade criativa total para pensar no tema e os detalhes do jogo.

Projeto 2: Multimidia

Um Projeto interativo para quem acessar. O desenvolvimento do meu projeto foi em base de java no aplicativo processing, e é basicamnete um app de aprendizagem de inglês para crianças pequenas.

Projeto 3: Web

Esse foi o ultimo projeto de experiência criativa, onde eu e meu grupo fomos instruidos a fazer um web site. Nosso trabalho deu frutos, conseguimos criar com sucesso um site voltado a educação de criança pequenas na área biológica.

Aulas de Lógica Matematíca

Jogo do Detetive

Em lógica Matematíca tivemos poucos trabalhos e grandes volumes de exercicios para avaliações. Entretanto, tivemos um trabalho bastante desafiador em que precisavamos criar um jogo de detetive em grupo.

*obrigações do projeto*

  • O trabalho devia ser feito em python e abordar o "import ttg".
  • Ele deveria apresentar proporções que remetessem ao conteúdo de Tabela Verdade.
  • Deveria apresentar uma historia sendo criada ou já existente.
  • Tinha de ser feita em grupos de até 4 pessoas.

Aulas de Fundamentos Ciberfísicos

Na diciplina de Ciberfísico tivemos apenas 2 trabalhos ao longo do semestre, ambos envolvendo sistemas operacionais e circuitos embarcados.

Sistema: BSD

Nesse projeto tivemos de pesquisar e apresentar em formato de video o sistema operacional BSD. O proprio é muito importante para o surgimentos de outras linguagens,como por exemplo o Unix.

O nosso projeto foi lançado no youtube por um dos nossos colegas e que infelizmente hoje eatá privado.

Modulo UBS

Esse trabalho foi feito no aplicativo Wokwi e tem como função desenvolver um sistema embarcado. Em nosso projeto, desenvolvemos um sistemas capaz de se contectar a internet e informar a temperatura, humidade e niveis sonoros de 3 Unidades basicas de saúde espalhadas pela cidade.
